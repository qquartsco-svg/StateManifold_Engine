# StateManifoldEngine 현재 상태 요약

## 🎯 엔진이 하는 일

**StateManifoldEngine**은 여러 난제 엔진의 붕괴 영역 선언을 겹쳐서 **퍼텐셜 우물 공간**을 형성하는 메타 엔진입니다.

### 핵심 기능

1. **상태 공간 구축** (`build_state_space`)
   - 여러 UP 엔진의 `SearchBias`를 차원으로 받아서 상태 공간 형성
   - 유기적 연결 가중치 계산
   - 통합 붕괴 영역 식별

2. **값이 공간을 통과** (`flow_through_space`)
   - 값이 여러 난제의 붕괴 조건을 동시에 고려하여 공간의 결을 따라 흐름
   - 흐름 에너지, 형태 보존도, 안정성 계산

3. **생명 유지 메커니즘** (`maintain_life`) ⭐ v0.2.0 신규
   - 최소 에너지로 상태 공간을 '살아있는' 상태로 유지
   - 열 잡음 + 미세 플럭추에이션 모사
